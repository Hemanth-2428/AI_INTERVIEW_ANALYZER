# 🤖 AI Interview Analyzer

An AI-powered web application that helps analyze interview responses and provide intelligent feedback to improve performance.

---

## 🚀 Features

* 🎤 Analyze interview answers in real-time
* 🧠 AI-based feedback and suggestions
* 📊 Performance insights and evaluation
* 💬 Easy-to-use interface built with Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI / LLM APIs
* NLP techniques

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/AI_INTERVIEW_ANALYZER.git
```

2. Navigate to the project folder:

```
cd AI_INTERVIEW_ANALYZER
```

3. Create virtual environment:

```
python -m venv venv
```

4. Activate environment:

```
venv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the App

```
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file and add your API keys:

```
OPENAI_API_KEY=your_api_key_here
```
